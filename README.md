## Vim Editor

-HOW TO USE VIM-
  
```bash
vim <FileName.FileType> // <FileName.FileType> 파일을 vim 에디터로 열기
```

- 파일을 처음에 열면 명령모드로 진입
- 입력 하려면 입력모드로 진입해야 함
  - 키보드에서 I 및 Insert 입력
- 명령모드 진입은 키보드에서 ESC키 누름
  - 명령모드 커맨드
    - 저장하기 > :w Write
    - 나오기 > :q Quit
    - 저장하고 나오기 > :wq Write and Quit
